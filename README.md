## Запуск

```shell
#Получение токена
./space-web get-token --login=your_login --password=your_password
#или
php space-web get-token --login=your_login --password=your_password

#Добавление домена
./space-web add-domain --domain=your_domain --token=your_token
#или
php space-web add-domain --domain=your_domain --token=your_token
```
